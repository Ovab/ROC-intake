De game

Minecraft is een spel waar alles om je heen blokkerig is en je alles kan doen wat je wilt, dit heet ook wel een sandbox game (link). Er zijn 2 Gamemodes: 1 "creative", hier heb je alle blokken en kan je vliegen 2 "survival", hier moet je proberen te overleven en zelf alle spullen verzamelen.

Kopen

Je kan de Minecraft java edition kopen voor 25 euro met deze link de java edition is de beste want daar kan je je eigen mods maken om de game aan te passen hoe je wil door het in Java te maken

Verhaal

Hoewel minecraft niet een specifiek verhaal heeft is er veel speculatie over ook zijn er 3 eindbazen die mogelijk hinten naar een verhaal zeker als met de "achievements" die je krijgt als je de baas verslaat
